# QGIS Plugin - SGU API

Hämtar data från Svenska geologiska undersökningens (SGU) API databas med val av datamängd, datasamling, koordinatreferenssystem (CRS), and bounding box (BBOX). Bekräftad funktionalitet på QGIS 3.40.11 

Fetches data from The Geological Survey of Sweden's (SGU) API database with choice of dataset, collection, coordinate reference system (CRS), and bounding box (BBOX). Confirmed functionality on QGIS 3.40.11
